For each set of information added (with attributes name, stock amount, weight), a unique ID is created. In order to delete or amend information, this ID needs to be referenced. This could be done by clicking "List" which provides the ID tag for each entry. This design choice was made for scalability, and it was inspired by the Git log and Git Checkout functionality.
